# Legal Boundaries

ORMB is a technical demo and portfolio project only.

## Prohibited Claims

Do not represent ORMB as:

- A launched stablecoin.
- A public RMB stablecoin.
- A regulated financial product.
- A custody product.
- A money transmission service.
- A retail trading token.
- A payment product ready for production use.

## Prohibited Activity

The MVP must not:

- Accept real customer funds.
- Process real USDT deposits.
- Process real RMB payments.
- Custody user assets.
- Provide redemption for real-world value.
- Deploy to mainnet without explicit human approval.

## Allowed Demo Activity

The MVP may:

- Use testnet contracts.
- Use mock deposit data.
- Use deterministic demo companies.
- Use fixed demo FX assumptions.
- Simulate operational approval and reconciliation flows.

## Required Language

Public documentation and demo materials must describe ORMB as a testnet-first technical demo, not a live financial product.
